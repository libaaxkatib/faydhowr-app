Settings Module — V1

Settings Dashboard · Company · Service · Store · Payment · Notification · Security · Numbering · Language · Roles · System Info. Admin only — Sales and Accountant have no access. Settings change system configuration only and never modify historical records. All changes are logged. Refinements: Global Search · Unsaved Changes Protection · Restore Defaults · Settings History · Maintenance Mode. Auth, Customer App, Dashboard, Customers, Bookings, Quotations, Orders, Payments, and Reports unchanged.
